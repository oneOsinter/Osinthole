🛡️ Анонимность & OPSEC
🔹 VPN (Mullvad, ProtonVPN, Express) 🔹 Tor + Bridges 🔹 Анонимные почты (ProtonMail, Tutanota)

🌐 VPN обязательно

Используй платные VPN без логов. Не экономь на безопасности. Kill-switch и DNS-защита.

🧅 Tor максимум

Для особо чувствительных задач. Не открывай личные аккаунты. Используй Tails OS.

📧 Почта временная

Для регистраций — Guerrilla Mail, 10minutemail. Для постоянной — ProtonMail без номера.

🔑 Пароли менеджер

Bitwarden / KeePass. Уникальные пароли для каждого сервиса. Двухфакторка (TOTP).

🕵️ Браузер Firefox + дополнения

uBlock Origin, NoScript, CanvasBlocker, ClearURLs. Отключи WebRTC.

💳 Оплата крипто / предоплата

Используй Monero или подарочные карты. Не привязывай личные карты.

📌 Правило нулевого доверия: никогда не используй реальные данные. Вся деятельность — через изолированную среду (виртуалка / live-usb).
